SMTP por Gmail
Ve a la configuración de tu cuenta de Google.
Busca "Seguridad" y activa la "Verificación en dos pasos".
Una vez activa, busca la opción "Contraseñas de aplicaciones".
Genera una nueva contraseña para una aplicación (puedes llamarla "Laravel Elan Spa") y copia el código de 16 dígitos que te proporciona

MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=tu-correo@gmail.com
MAIL_PASSWORD="la-contraseña-de-16-digitos"
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="tu-correo@gmail.com"
MAIL_FROM_NAME="ELAN SPA"

Explicación de cada línea:
MAIL_MAILER=smtp: Le dice a Laravel que use el protocolo estándar SMTP.
MAIL_HOST: La dirección de tu servidor de correo (ej. smtp.mailgun.org, smtp.hostinger.com).
MAIL_PORT=587: El puerto de conexión. 587 es el más común para la encriptación TLS. Otros comunes son 465 (SSL) o 2525.
MAIL_USERNAME: Tu nombre de usuario para el servidor de correo, que usualmente es tu dirección de correo completa.
MAIL_PASSWORD: La contraseña de tu cuenta de correo. Es recomendable ponerla entre comillas si contiene caracteres especiales.
MAIL_ENCRYPTION=tls: El tipo de encriptación. tls es el estándar moderno y preferido.
MAIL_FROM_ADDRESS: La dirección de correo que aparecerá como remitente.
MAIL_FROM_NAME: El nombre que aparecerá como remitente. Usar ${APP_NAME} tomará el nombre de tu aplicación definido en el mismo archivo .env.
